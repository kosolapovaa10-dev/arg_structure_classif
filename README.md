# arg_structure_classif
Проект посвящён автоматической классификации отношений между элементами аргументативной структуры в медицинских текстах на нескольких языках (английский, испанский, французский, итальянский).

Модель обрабатывает медицинские случаи и выявляет отношения типа **Support** и **Attack** между компонентами аргументации (например, Premise, Claim).

Цели проекта
- Подготовка многоязычного датасета для классификации отношений между компонентами аргументации.
- Перенос отношений из английского сабсета на другие языки (ES, FR, IT) с помощью эмбеддингов.
- Обучение модели XLM-RoBERTa для задачи классификации отношений.
